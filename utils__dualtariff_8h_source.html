17 /** @brief Config parameters for overriding a load
18  * @details This class allows the user to define when and how long a load will be forced at
19  * full power during off-peak period.
20  *
21  * For each load, the user defines a pair of values: pairForceLoad => { offset, duration }.
22  * The load will be started with full power at ('start_offpeak' + 'offset') for a duration of 'duration'

24  * - if the offset is negative, it's calculated from the end of the off-peak period (ie -3 means 3 hours back from the end).
25  * - to leave the load at full power till the end of the off-peak period, set the duration to 'UINT16_MAX' (somehow infinite time)
26  */
