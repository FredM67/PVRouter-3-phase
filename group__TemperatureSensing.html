Detailed Description

Functions used for temperature monitoring and processing.

This group contains functions that handle temperature sensing using DS18B20 sensors. It includes reading temperature data, filtering invalid readings, and updating telemetry with valid temperature values. The temperature data can also influence load diversion decisions.

Note
The system supports multiple sensors and stores temperature values as integers multiplied by 100 for precision.

Represents a buffer for storing sensor data.

The ScratchPad type alias is used to define a buffer of 9 bytes, which is typically used to store data read from a DS18B20 temperature sensor's scratchpad memory.

  • The scratchpad memory contains temperature data, configuration settings, and a CRC byte.
  • This buffer is used during temperature reading and validation processes.
